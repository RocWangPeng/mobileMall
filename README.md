# koa+vue移动电商

---
移动电商系统，包括首页展示，类别展示，购物功能，注册登录，积分系统，查找页面，后台接口设置，数据库采用在线mlab在线的mongo数据库

## 主要技术栈列表
- Vue
- Vuex
- Router
- Vant
- Node
- Koa2
- Mongoose
- MongoDB
